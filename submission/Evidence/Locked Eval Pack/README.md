# Locked Eval Pack Pointer

The frozen Week 7 evidence pack used by the thesis is stored at:
- `reports/week7/locked_eval_pack/week7_norm_only/`

## Key files to inspect
- `README.md` - protocol summary and contents
- `DATA_STATS.md` - authoritative split counts
- `RUN_CONFIG_SNAPSHOT.md` - final run settings and threshold metadata
- `tables/` - aggregate result tables
- `figures/` - ROC and histogram figures
- `error_cases/` - curated false-positive and false-negative examples
- `ENV.txt` - environment snapshot

This submission folder uses a pointer rather than duplicating the entire locked pack so there is only one source of truth for the final metrics and counts.
